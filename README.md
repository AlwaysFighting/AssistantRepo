## 고용인앱
📌 목표 : Flutter 르 사용해서 IOS 와 안드로이드 앱을 동시에 작업한다.

## 데이터베이스 구조도
🎾 전체 구조도는 다음고 같다
 - 데이터베이스를 저장한 시간 구조도
 - DropDown 에 표시할 데이터베이스 (기기 종류)
 - 기기의 값을 저장할 데이터베이스
